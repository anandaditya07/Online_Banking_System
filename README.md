Online Banking System

Overview

This repository contains the source code for a simple online banking system, providing basic functionality for customers to manage their accounts and transactions.

Features

- User registration and login functionality
- Account management (view balance, transaction history)
- Fund transfer (internal)
- Transaction tracking
- Admin panel for managing user accounts and transactions
- Basic password encryption and authentication

Technologies Used

- Java (SE 17 or higher)

Getting Started

Prerequisites

- Java Development Kit (JDK) 17 or higher

Installation

1. Clone the repository: git clone (link unavailable)
2. Install MySQL Community Server and create a database
3. Update DatabaseConnection.java with your database credentials
4. Compile and run the project using your preferred Java IDE
